# Task Management API


A simple Django REST Framework API for managing tasks.

## Features
- Create tasks
- List tasks
- Update tasks (optional)
- Delete tasks (optional)

## Endpoints
| Method | Endpoint        | Description        |
|--------|------------------|--------------------|
| GET    | /api/tasks/      | List all tasks     |
| POST   | /api/tasks/      | Create new task    |
